**CHARMIX** 

**Personalized Beauty & Skincare Recommendation Platform**

CHARMIX is a full-stack desktop application that helps users discover beauty products and receive personalized skincare routine recommendations based on their skin profile. The platform also includes product browsing, search/filtering, checkout flow, and an admin dashboard for managing inventory and orders.

**User Features** 
* User registration and login
* Interactive skincare questionnaire
* Personalized routine recommendations based on user inputs
* Product catalog browsing
* Search and multi-filter product discovery
* Add to cart and checkout workflow
* Order tracking and history
* Admin mode - Inventory management, User access, Report generation and analytics.
